## Come usarlo (locale)
1. Estrarre la cartella.
2. `npm install`
3. `npm run dev`
4. Aprire `http://localhost:5173`

## Personalizzazione rapida
- Sostituisci l'endpoint del form in `src/App.jsx` (variabile `endpoint` nella pagina Contatti) con il tuo endpoint Formspree o backend.
- Aggiorna email/telefono nel Footer.
- Sostituisci la sezione "hero-image" con una vera foto (in `public` e <img> se preferisci).
